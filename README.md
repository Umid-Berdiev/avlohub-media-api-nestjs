## Description

Media API project made with [Nest](https://github.com/nestjs/nest) framework & TypeScript.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm start

# watch mode
$ pnpm start:dev

# production mode
$ pnpm start:prod
```
